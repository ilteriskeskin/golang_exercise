# golang_exercise

**Go Install**
* [Download Go](https://golang.org/dl/go1.15.5.linux-amd64.tar.gz)
* tar -C /usr/local -xzf go1.15.5.linux-amd64.tar.gz
* export PATH=$PATH:/usr/local/go/bin
* go version
